Zero-shot Spatial Reasoning with LLM

2024

  • Python
  • PyTorch
  • LLM
  • vLLM

Course project for COMP5214 (Advanced Deep Learning Architecture). We assessed the ability of spatial reasoning of LLM, which performed better than some basic vLLM models. The LLM could generate more humanoid actions than traditional algorithms. Moreover, LLM performed better in multi-agent scenarios with collision possibilities.
